# 带状疱疹疫苗惠民活动落地页

一个精美的疫苗活动推广落地页，采用高级深蓝紫+金色配色方案。

## 🎨 设计特点

- **高级配色**: 深蓝紫渐变 + 金色点缀
- **响应式设计**: 完美适配手机、平板、电脑
- **动态效果**: 倒计时、动画、悬停效果
- **易于阅读**: 大字体，适合中老年用户
